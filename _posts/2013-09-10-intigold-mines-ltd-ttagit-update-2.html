Intigold Mines Ltd. - TTAGIT Update

Vancouver, BC, September 10, 2013, Intigold Mines Ltd. (IGD - TSX.V and IDMNF - OTC QX) has received the following update from TTAGIT Social Networks Inc. ("TTAGIT"). Intigold owns a 51% interest in TTAGIT.

"TTAGIT is architecting an API which will complement its patent-pending revolutionary technology which will work in pulling all tweets from any URL whether they be shortened or not. This will allow users to see all tweets connected with a URL (web page). When a user opens the TTAGIT extension over a web page, if there has been any tweets of that URL, it will show in the feed. This will add content to TTAGIT's streams and show what people are talking about on URL's from Twitter thus helping to populate TTAGIT. This move is a start to bringing social networks into TTAGIT's architecture in order to interact with content relevant to a URL (web page). Users would then be able to respond back which would also tweet the response as an @Reply on Twitter. TTAGIT is also working on utilizing Twitter's card option which let's users attach a video, photo, gallery or a product description to a tweet. TTAGIT is also working on architecture that will allow users to interact more with Twitter from within TTAGIT, like the use of #Hashtags and @Replies within TTAGIT. "The company is very pleased with the current progression of TTAGIT and the upcoming development of Phat Pen (TTAGIT's Financial Application) and looks forward to implementing these changes to increase content over URL's using TTAGIT's engine" stated Leigh Akin , Creative Director of TTAGIT."

Developed and owned 51% by Intigold Mines Ltd., TTAGIT is a revolutionary online tool that lets you comment on everything on the internet from your browser without having to login to several accounts or register with a number of other websites. Designed to "tag" the moment, TTAGIT is the new face of commenting and social networking.

TTAGIT, the first open-forum discussion social network and comment management system that permits users to comment on all URLs with one login and allows you to talk over any website without leaving the page. TTAGIT is a commenting and social networking tool where members can engage in conversations and participate in debates on sites and portals that currently do not allow live streaming.
