NEWS RELEASE

Vancouver, BC, April 22, 2015, Intigold Mines Ltd.(IGD - TSX-V) announced today that it has received the following update from TTAGIT Social Networks Inc. ("TTAGIT"). Intigold owns a 51% interest in TTAGIT.

TTAGIT - Update

TTAGIT has pulled out of Beta and into production with the Twitter powered extension set. TTAGIT has released a mobile Action Extension on IOS8 Apple Mobile devices that works with ipad, ipod and iphone 5 and 6. TTAGIT also has the Chrome Classic as well as Safari which we have added for desktops.

TTAGIT has released a new website that is 100% Twitter Bootstrap to keep in relevance to our use of Twitter as a Mashup. The new website keeps simplicity and is responsive for Mobile use.

The new action extension lets users interact with Twitter through the share function built right into Apple's IOS system for a more user-friendly feel. Users can open the application while surfing the Internet and through the Tweets on the Share function bring up an overlay which shows and lets users interact with Twitter relevant to the URL the user is on. TTAGIT turns any URL into its own sound board.

The Desktop extensions let users interact with Twitter relevant to any URL by using the browsers use of extensions and add-ons on Safari and Google Chrome.

At this point, the only way to use TTAGIT on mobile is with Apple's IOS, but there may be further developments on other platforms as other platforms catch up to web standards.

Users can download extension from http://ttagit.com.

TTAGIT - de2our

TTAGIT has also made a name change on the proposed application for ticket sales The name has changed from de2our to Tickbit.

TTAGIT - Technology

TTAGIT is a proprietary technology which allows you to comment, talk and respond over any website (URL) using live feed while you are viewing the page without having to leave the page that you are on and go to a different website to view or post comments. This is what gives TTAGIT a competitive advantage.
