NEWS RELEASE

INTIGOLD MINES LTD. - TTAGIT Update

Vancouver, BC, April 21, 2016, Intigold Mines Ltd. (IGD - TSX-V) (OTC:IDMNF, - News) announced that it has received the following update from TTAGIT Social Networks Inc. ("TTAGIT"). Intigold owns a 51% interest in TTAGIT.

TTAGIT Update

TTAGIT is compiling all the scripts for its technology into one master script so as to offer out an API (Application Programming Interface). This will allow TTAGIT to not only port out to more platforms but also for Strategic Business Partnerships to make use of each other within the application. This process will be short, and the cleaned script will be replacing the platforms that are already available. The first stage is releasing it on iPhone which will be coming soon. Also, the new script will be usable on Android. We will push that to the Android store if we see interest in it.

TTAGIT – Technology

TTAGIT is a proprietary technology which allows Internet users to comment, talk and respond over any website (URL) using live feed. By using TTAGIT, users can stay on a webpage and post to their favorite social networks, without ever going to different website's to view or post comments. This is what gives TTAGIT its competitive advantage.
